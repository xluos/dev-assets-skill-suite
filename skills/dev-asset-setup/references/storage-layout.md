# Storage Layout

`<repo>/.dev-assets/<branch>/`

Core files:

- `overview.md`: 主需求摘要与目录导航
- `prd.md`: 产品需求、目标、范围、验收
- `review-notes.md`: 评审结论、争议点、action
- `frontend-design.md`: 前端页面、状态、交互、接口依赖
- `backend-design.md`: 接口、数据、兼容性、发布影响
- `test-cases.md`: 测试主流程、边界、回归
- `development.md`: 开发过程与 Git 自动同步块
- `decision-log.md`: 后续会反复引用的技术/产品结论
- `commits.md`: 提交记录
- `artifacts/`: 其他附件或中间产物
