使用说明
· 主视图是 proposal 卡片 —— 每张卡是一个"事项"，agent 已经把相关条目聚合好
· 每张卡片右上角 accept / reject 切换；展开 查看影响 entries 看具体动了哪些 entry / section / 文件
· 默认每张卡都是 accept。一眼扫完，不同意的点 reject 即可
· 漏网之鱼：底部"未提议"折叠区可看 agent 没聚合的 entry，里面提供单条"标 delete"按钮
· 审完点蓝色"导出 plan.json" → 文件下载到 Downloads → 告诉 agent 路径继续 apply（apply 永远先备份）
